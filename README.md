## 1. Deskripsi Aplikasi
Aplikasi ini berfungsi untuk mengelola data buku dengan fitur:
- Menambahkan data buku
- Menampilkan seluruh data buku
- Mengubah data buku
- Menghapus data buku
- Mengunggah file cover buku (JPEG/PNG)
Struktur aplikasi menerapkan OOP sederhana menggunakan Repository Class, sehingga kode lebih rapi, modular, dan mudah dipelihara.

## 2. Spesifikasi Teknis
**Teknologi yang digunakan:**
- PHP 8.x (Native)
- MySQL / MariaDB
- PDO (PHP Data Objects)
- HTML dasar
- XAMPP
